class io_swix.CSVFile[source]¶

A simple class to represent a CSV file. It has two members, “header” and “data”. “header” is a list of string and “data” is a matrix.

>>> var x:matrix = rand(N:(3, 3))
>>> csv = CSVFile(x, ["1", "2", "3"])
>>> write_csv(csv, filename:"/path/to/file")
>>>
>>> # one of the following options
>>> var csv:CSVFile = read_csv("/path/to/file")
>>> var x:matrix = read_csv("/path/to/file").data

io_swix.read_csv(filename)[source]¶

Reads a CSV.

Parameters:
  • filename (String) – The file to read from. The file read from the directory containing swix/.
  • rowWithoutMissingData (Int) – A row that has complete data. From this row, we infer if that particular row is categorical or numerical for the entire CSV. If there are N categories, we assign indicies 0 through N-1.
  • header_present – If false, the first row will be included as part of the dataset. It default to true in which case the header will also be returned.
Return type:

ndarray. The contents of the csv.
